# 比亚迪半导体销售情报手册

## 📖 项目简介

这是一个专业的网页格式销售手册,用于支持比亚迪半导体的销售团队。手册采用咨询公司培训手册的专业布局,包含详细的产品信息、客户分析、竞争情报和销售话术。

## 🎨 设计特点

### 视觉风格
- **品牌色系**: 比亚迪经典红色主题 (#e60012)
- **页面尺寸**: A4横版 (297mm × 210mm)
- **字体**: 微软雅黑、黑体等中文友好字体
- **布局**: 专业咨询风格,清晰的信息层级

### 核心元素
- **渐变标题栏**: 红色渐变背景,右侧斜角设计
- **封面/尾页**: 上下分割设计,底部红色区域配有logo和口号
- **数据可视化**: 卡片式数据展示,表格斑马纹设计
- **信息框**: 多种颜色编码的信息框(策略、洞察、警告等)

## 🚀 使用方法

### 在线浏览
1. 直接在浏览器中打开 `index.html` 文件
2. 推荐使用Chrome、Edge或Firefox等现代浏览器
3. 可以使用浏览器的打印功能导出为PDF

### 打印设置
如需打印或导出PDF:
1. 打开浏览器的打印对话框 (Ctrl+P 或 Cmd+P)
2. 目标打印机选择"另存为PDF"
3. 纸张尺寸选择 A4 横向
4. 边距设置为"无"或"最小"
5. 确保勾选"背景图形"选项

## 📋 内容概览

### 包含的产品模块
1. **IGBT** (绝缘栅双极型晶体管)
   - 市场环境分析
   - 客户分级矩阵
   - 竞争格局对比

2. **车载摄像头模组**
   - 智能驾驶应用
   - 客户决策链分析
   - 竞品深度对标

3. **车规级LED**
   - 照明系统应用
   - 市场机会识别
   - 竞争策略定位

4. **MCU** (微控制单元)
   - 产品矩阵展示
   - 客户需求分析
   - 核心竞争优势

5. **销售场景实战**
   - 客户砍价应对
   - 竞品挖角防御
   - 独家供应谈判

### 分析框架 (4C)
每个产品都采用统一的4C分析框架:
- **C1: Context** - 市场环境情报
- **C2: Customer** - 客户情报分析
- **C3: Competitor** - 竞争情报研究
- **C4: Company** - 自身能力展示
